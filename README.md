#### Why ENS?

**Ethereum Name Service (ENS)** provides a secure and decentralized way to map human-readable names like `alessandro-cavaliere.eth` to machine-readable identifiers such as Ethereum addresses or IPFS content hashes. This system ensures that my website remains accessible and resistant to censorship while reflecting my commitment to the principles of decentralization, very dear to me. My domain interacts directly with the **ENS Registry contract** at [0xD4416b13d2b3a9aBae7AcD5D6C2BbDBE25686401](https://etherscan.io/address/0xD4416b13d2b3a9aBae7AcD5D6C2BbDBE25686401), which is responsible for managing the ownership and resolution of `.eth` domains on the Ethereum network. 

#### Why IPFS?
**The InterPlanetary File System (IPFS)** is a distributed file system that allows for the storage and sharing of data across a decentralized network. Unlike traditional hosting methods, IPFS makes the website resilient to failures, as content is stored across multiple nodes, ensuring availability even if some nodes go offline. This aligns with the decentralized nature of the **blockchain** and enhances the security and permanence of my portfolio. We don't want our provider to crash. ğŸ˜‰
